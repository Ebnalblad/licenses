Open Licenses Service

A web service that lists open (http://opensource.org/, http://opendefinition.org/) licenses. The listings are provided in json and the full set of open-source and open knowledge (data and content) licenses are provided.

How do I use it?

List of licenses are available in the following groups (groups are JSON list of license hashes/dictionaries):

  • all_alphabetical – all the licenses (in alphabetical order)
  • ckan_original – CKAN license set.
  • ckan_canada – CKAN Canada set
  • ukgov – UK Gov license set

Licenses format

Licenses are provided as JSON dictionaries, for example:


{
  "date_created": "2010-03-09T11:41:18.108194", 
  "family": "", 
  "id": "odc-odbl", 
  "is_okd_compliant": true, 
  "is_osi_compliant": false, 
  "maintainer": "", 
  "status": "active", 
  "tags": [], 
  "title": "Open Data Commons Open Database License (ODbL)", 
  "url": "http://www.opendefinition.org/licenses/odc-odbl"
}

Python package

The data together with a simple API and ancillary code are also available as a python package: http://pypi.python.org/pypi/licenses.

The Licenses service is run by the Open Knowledge Foundation. Source Code.
